# Модуль с собранной миссией 09.01 для shadowrun

Модуль содержащий официальную миссию 09.01. Основой являются официальные переводы миссий на русский язык, с корректировками из оригинальных миссии на английском.

Характеристики персонажей в модуле могут отличаться от характеристик описанных в миссии.

Почти все арты сгенерированы через бесплатные AI, а подходящие карты взяты из интернета.

Автор модуля не претендует на авторство контента. Это просто сборка всего в кучу, для удобного ведения игры. Распростряняется бесплатно.
